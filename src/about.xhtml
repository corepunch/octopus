About Octopus

An open markdown blog for writers and readers.

What is Octopus?

Octopus is a minimalist, open blogging platform where anyone can write, share, and follow the writers who inspire them. Posts are written in Markdown and rendered beautifully in your browser — no apps required.

Who is it for?

Octopus is for developers, thinkers, and anyone who values clear writing over algorithmic noise. There are no ads, no tracking pixels, and no engagement-bait notifications — just words.

Open source

Octopus is fully open source. The source code is available on GitHub. Contributions, bug reports, and feature requests are welcome.

Built with

  • Handlebars — client-side templates
  • Marked — Markdown rendering
  • Appwrite — backend and authentication
  • GitHub Pages — static hosting
